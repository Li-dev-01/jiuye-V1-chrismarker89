## 🚀 技术架构

### 前端技术栈
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI库**: Ant Design 5.x
- **状态管理**: Zustand + Context API
- **路由**: React Router v6
- **样式**: CSS Modules + 响应式设计
- **AI集成**: 自定义AI服务层

### 后端技术栈
- **运行时**: Cloudflare Workers
- **框架**: Hono.js
- **数据库**: SQLite (D1)
- **认证**: JWT + 权限系统
- **API**: RESTful API设计

### AI集成架构
