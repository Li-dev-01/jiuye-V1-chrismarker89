### **角色功能分解**
```
🛡️ 超级管理员 (SUPER_ADMIN)
├── 紧急控制: 项目关闭/恢复
├── 安全防护: DDoS防护、IP封禁
├── 威胁监控: 实时安全分析
└── 事件处理: 安全事件记录

👨‍💼 管理员 (ADMIN)  
├── 用户管理: 账号、权限管理
├── 内容管理: 问卷、心声、故事
├── 系统配置: 规则、参数设置
└── 数据分析: 统计、导出功能

👨‍⚖️ 审核员 (REVIEWER)
├── 问卷审核: 问卷响应审核
├── 心声审核: 用户心声审核  
├── 故事审核: 用户故事审核
└── 审核统计: 工作量和效率
```
