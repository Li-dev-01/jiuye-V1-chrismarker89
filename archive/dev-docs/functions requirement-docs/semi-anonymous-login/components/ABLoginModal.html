半匿名身份验证

请输入您的A+B组合以验证身份

A值为11位数字，通常使用手机号码
B值为4位或6位数字，作为您的身份密码
勾选后将在本设备记住您的身份信息，下次无需重新输入
💡

提示

如果您之前在问卷或故事墙中使用过A+B身份标识，请输入相同的组合。

系统不会存储您的原始A+B值，只会生成加密的身份标识。

什么是A+B组合？
A值（身份标识）
  • 11位数字，通常使用手机号码
  • 作为您的主要身份标识
  • 示例：13812345678
B值（身份密码）
  • 4位或6位数字
  • 作为您的身份验证密码
  • 示例：1234 或 123456
安全说明
  • 系统只存储加密后的身份标识
  • 原始A+B值不会被保存
  • 每次登录都需要输入完整的A+B组合
