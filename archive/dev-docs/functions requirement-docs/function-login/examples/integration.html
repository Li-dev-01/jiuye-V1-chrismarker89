管理员登录功能集成示例

1. 基础集成

最简单的集成方式，只需要引入CSS和JS文件，然后创建一个容器元素。

<!-- 引入样式文件 --> <link rel="stylesheet" href="components/AdminLoginPage.css"> <!-- 创建登录容器 --> <div id="login-container"></div> <!-- 引入脚本文件 --> <script src="services/authService.js"></script> <script src="components/AdminLoginPage.js"></script>

2. 自定义配置

通过配置选项自定义登录行为，包括成功/失败回调、重定向控制等。

const loginPage = new AdminLoginPage({ container: '#custom-login-container', redirectEnabled: false, // 禁用自动重定向 onSuccess: (user) => { console.log('登录成功:', user); // 自定义成功处理逻辑 showUserInfo(user); }, onError: (error) => { console.error('登录失败:', error); // 自定义错误处理逻辑 showErrorMessage(error); } });

3. API使用

直接使用认证服务API进行登录验证和权限检查。

// 直接调用登录API const result = await AuthService.adminLogin('admin1', 'admin123'); if (result.success) { console.log('登录成功:', result.data.user); } // 检查权限 const hasPermission = AuthService.hasPermission('content_review'); console.log('是否有内容审核权限:', hasPermission); // 获取当前用户 const currentUser = AuthService.getCurrentUser(); console.log('当前用户:', currentUser);
