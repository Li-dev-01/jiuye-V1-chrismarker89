## 📊 性能测试

### 加载性能

使用 Chrome DevTools (移动模式):

1. 打开 DevTools
2. 切换到 Mobile 模式
3. 选择设备 (iPhone 12)
4. 运行 Lighthouse 测试

**目标指标**:
- [ ] Performance Score > 80
- [ ] First Contentful Paint < 2s
- [ ] Largest Contentful Paint < 3s
- [ ] Time to Interactive < 4s
- [ ] Cumulative Layout Shift < 0.1

### 网络性能

测试不同网络条件:

- [ ] **Fast 3G** (1.6 Mbps)
  - 页面可用时间 < 5s
  - 交互响应正常

- [ ] **Slow 3G** (400 Kbps)
  - 页面可用时间 < 10s
  - 基本功能可用

- [ ] **Offline**
  - 显示离线提示
  - 缓存内容可访问
